# Example: Hybrid Node (Human + AI)

## Scenario

Bob uses a Claude-powered companion AI to help him evaluate a vendor contract. Their joint session is logged as a Hybrid Node — the AI's recommendation and the human's confirmation are both recorded in the same chain.

## How HNBP-CORE Is Used

1. The companion AI initializes a session log.
2. The AI's recommendation is appended as a record (including confidence score).
3. Bob's explicit human confirmation is appended.
4. The complete decision trail is signed as a hybrid actor.

## Why This Matters

In disputes, both parties can prove: "The AI recommended X, and the human confirmed Y, at this exact time." Neither party can retroactively change their position. The log is the ground truth.
