About HNBP-CORE

HNBP-CORE is a universal, tamper-evident, append-only witness protocol for humans, AI agents, and systems — with optional Bitcoin anchoring.

Record Structure
{
  "index":     integer (0, 1, 2, ...),
  "timestamp": ISO 8601 string,
  "data":      any JSON-serializable value,
  "prev_hash": hex string | null (index 0 only),
  "hash":      SHA-256 hex digest
}
Hashing Rule
hash = SHA256(
  String(index) +
  String(timestamp) +
  JSON.stringify(data, sorted_keys) +
  String(prev_hash)   // "null" for index 0
)
Validity Rules
  • Non-empty JSON array
  • Indices exactly sequential: 0..n
  • records[0].prev_hash === null
  • records[i].prev_hash === records[i-1].hash for i > 0
  • Recomputed hash matches stored hash for every record
  • All timestamps are valid ISO 8601
